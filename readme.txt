Rafał Prusak - praca inżynierska
	
Program dokonujący detekcji oraz klasyfikacji pojazdów na podstawie obrazu video z kamery.

Program wykonany w języku Python(v3.4), uruchamiany na systemie ubuntu 14.04.

Wymagania:
- biblioteka opencv 3.0 
	instrukcja instalacji dla ubuntu: 
	http://milq.github.ioinstall-opencv-ubuntu-debian
- biblioteka pygtk+
